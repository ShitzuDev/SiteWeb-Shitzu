# 🌐 Site Web "ShitzuDev" 



Bienvenue sur mon portfolio en ligne ! Ce site a été conçu en HTML, CSS et JavaScript pour présenter mon profil, mes projets, mes réseaux sociaux et bien plus encore. 🚀

## 📜 Description

Ce site est un portfolio personnel qui me permet de partager mes compétences, mes projets passés et mes liens vers mes profils sociaux. Vous y trouverez des informations à propos de mes réalisations, ainsi que des liens vers mes projets open source et mes plateformes professionnelles. 💻

## 🚀 Fonctionnalités

- **Page d'accueil** : Un aperçu de mon profil avec une brève introduction. 👤
- **Mes projets** : Une section dédiée où vous pouvez découvrir mes projets réalisés. 📁
- **Mes réseaux sociaux** : Des liens vers mes différents profils sociaux, afin de mieux me suivre et échanger. 🔗
- **Interactivité** : Grâce à l'utilisation de JavaScript, certains éléments du site sont interactifs pour une expérience utilisateur améliorée. ✨

## 🛠️ Technologies utilisées

- **HTML** : Structure de la page et contenu. 🏗️
- **CSS** : Mise en forme, design et responsivité. 🎨
- **JavaScript** : Ajout d'interactivité (animations, effets, etc.). ⚡
